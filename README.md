# Market Maker Coin Scanner

A Streamlit application to scan for market making opportunities on Lighter DEX.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Features
- Volume filtering
- Spread calculation and filtering
- Volatility analysis
- Sweet Spot detection
